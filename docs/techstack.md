## Tech Stack

Use the latest stable versions available.

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Supabase
  - Auth
  - PostgreSQL
  - Storage
  - Edge Functions
- Drizzle ORM
- React Hook Form
- Zod
- dnd-kit
- Stripe
- Resend
- PostHog
- Vercel

Follow modern Next.js 16 best practices.

Use:

- Server Components by default
- Client Components only when necessary
- Server Actions instead of API routes whenever possible
- React Server Actions
- Route Handlers only when required
- Suspense where appropriate
- Optimistic UI when it improves UX
- Dynamic metadata
- Image optimization
- Proper loading.tsx and error.tsx files
- Strict TypeScript
